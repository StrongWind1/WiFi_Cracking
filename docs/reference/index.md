# Reference

This section contains technical reference material for WiFi security analysis.

- [EAPOL-Key Frame Format](eapol-frame.md) -- Byte-level structure of EAPOL-Key
  frames and the Key Information bitfield used to identify handshake messages.
- [Capture Requirements](capture-requirements.md) -- Minimum fields and frames
  needed to extract crackable hashes from PSK, EAP, and WEP captures.
- [Hash Extraction Reference](hash-extraction.md) -- Detailed format specifications
  for WPA hash lines, hashcat modes, and EAP hash formats.
- [Gap Table](gap-table.md) -- Known gaps between 802.11 specifications, hcxtools
  implementation, and hashcat support.
- [Glossary](glossary.md) -- Definitions of key terms used throughout this site.
